# GlamourCheck

GlamourCheck is a Dalamud plugin for Final Fantasy XIV that helps track gear collected for glamour.

## Features

- Tracks gear across inventory, equipped gear, armory chest, glamour dresser, armoire, chocobo saddlebag, and retainers.
- Shows small collection markers on gear icons.
- Shows duty loot completion counters in duties and in the Duty Finder.
- Provides hoverable loot grids grouped by role and gear slot.
- Supports trying on duty gear sets from the Duty Finder popup.
- Uses Garland Tools data, with local seeding for duties with incomplete loot data.

Open settings with:

```text
/glamourcheck
```

## Build

```text
dotnet build GlamourCheck.sln -c Debug
```
